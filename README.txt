This is a joint project of GECE (Group of Electrnoics and Computer Engineers) 
Member of which are Satyam Dutta, Siddharth Sivakumar, Shivam Agarwal and Krishna Ojha

It is a Local Network Based website which can be used for sharing of different file types like Audio, Video, txt etc
among users on the same network. It is hosted on a local server and has been assigned a static IP for the convenience 
of users.

It contains all the designing libraries of MDBootstrap, Fontawesome and Required Google Fonts downloaded to keep it completely 
independent of Internet

#== IMPORTANT ==#
To run this Website or the contents of this repository you'll have to host it on either a Local or Cloud based server.
